# Maintenance

Updating cookies
- Add/remove rows in `html/cookie-policy.html` table
- Update descriptions and durations as needed
- Reflect any new categories in the banner/preferences copy

Versioning
- Update Last Updated/Effective Date on policy page
- Keep changes in git for audit trail

Exporting consent logs
- This package stores consent client-side only.
- If you need server-side logs, send the cookie contents to an API when users change preferences and store with timestamp + IP (consider privacy laws).
